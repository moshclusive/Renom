Advanced Features

Powerful tools for professional users

Work with Sub-folders

Process files in all sub-folders automatically. Rename entire directory trees in one operation.

Smart Conditions (IF/THEN)

Apply rules only to specific file types. For example: "IF file is JPG THEN add date to name".

Safe Undo System

Made a mistake? Restore original names instantly. Files go to Renom's trash, not system recycle bin.

Zip & Organize

Automatically create ZIP files from groups of files, or move files into organized folders.

Photo Metadata (EXIF)

Rename photos using camera data: "Date Picture Taken", resolution, camera model, and more.

Music Tags (ID3)

Rename MP3 files using artist name, song title, album, or track number from file tags.

File Properties

Use Windows file info: video length, image dimensions, document author, title, and 100+ more attributes.

Change Timestamps

Modify file creation date, last modified date, and last accessed date.

Change Attributes

Set files as Hidden, Read-only, or Archive. Control file visibility and protection.

Import from List

Rename files using names from a text file or CSV spreadsheet. Perfect for client deliveries.

Fill Number Gaps

Automatically detect missing numbers in sequences and reorganize files to fill gaps.
